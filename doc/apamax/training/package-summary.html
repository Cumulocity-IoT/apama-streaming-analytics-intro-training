Package apamax.training

Event summary
AlertSend an alert via DES to be diplayed on the alerts dashboard.
RegisterRobotThis event is send by a robot which its (dynamically generated) ID when it starts up.
RequestRobotStatusSend this event for the robot manager to reply with the status of all the robots.
RobotDataRequestRequest a summary of the current data of a given robot from the robot manager.
RobotDataSummaryThis event is returned in response to a RobotDataRequest and contains the current data of the robot.
RobotErrorEvent sent by the robot when it detects an internal error.
RobotStatusAn event sent out by the robot manager whenever the status of a robot changes or in response to a RequestRobotStatus event.
TemperatureReadingTemperature readings taken periodically from the sensor on a robot. Will be sent periodically in normal running of the robot.
