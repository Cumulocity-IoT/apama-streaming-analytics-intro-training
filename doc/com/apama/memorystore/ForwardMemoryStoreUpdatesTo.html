com.apama.memorystore
Event ForwardMemoryStoreUpdatesTo

Requests Scenario Service updates for an exposed MemoryStore to be forwarded to the specified Correlator context.

Note that this is inefficient and does not implement proper flow control. Only use for testing or debugging.
See Also:
com.apama.memorystore.Schema#exposeMemoryView - Only tables that are configured to expose their contents will be affected by this event.
