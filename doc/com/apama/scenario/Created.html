com.apama.scenario
Event Created

Notifies all interested clients that a new instance of a specific scenario has been created. The event provides the owner (user), initial state, and initial values for all input fields and all output fields.

See also: Create()

Direction: From the correlator to the client.

Channel: .Data and .Data.Raw
